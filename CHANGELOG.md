# Change Log

All notable changes to the "Copy Code Ref for AI" extension will be documented in this file.

## [0.1.5]

### Added

- Add `Copy Code: Show Local Shortcut Usage Stats`.
- Track local usage counts for copy, open AI coding tool, and send-to-terminal shortcut commands.
- Document the local usage stats feature in English and Chinese.

### Changed

- Format Codex references as clickable Markdown links, for example `[/path/to/file.ts:137](/path/to/file.ts:137)`.
- Format Claude Code and terminal references as `@file#line` or `@file#start-end` mentions.

## [0.1.0 - 0.1.4]

### Added

- Copy selected code locations as file and line references.
- Open Codex or Claude Code with the selected reference prefilled.
- Send references to the active terminal.
- Select the target AI coding tool from the Command Palette.
